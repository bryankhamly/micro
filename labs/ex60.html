Exercise 60: ELISA

Enzyme-Linked Immunosorbent Assay - Core Concepts

Key Concepts for the Test

Key Terminology

Term Definition
Antigen Molecule that triggers immune response; recognized by antibodies
Antibody Protein (immunoglobulin) produced by B cells that binds antigens
Immunoassay Test using labeled antibodies to detect antigen-antibody interactions

Two Types of Immunoassays

Type Detects Example
Direct Unknown ANTIGENS in sample Detecting bacteria in throat swab
Indirect Specific ANTIBODIES in patient serum HIV antibody test (this lab)

Indirect ELISA Steps (HIV Test)

  1. Coat wells with HIV antigen
  2. Add patient serum - If anti-HIV antibodies present, they bind
  3. Wash - Remove unbound material
  4. Add secondary antibody - Anti-IgG linked to enzyme (peroxidase)
  5. Wash again
  6. Add substrate - Enzyme produces COLOR if antibodies were present

Interpreting Results

Result Color Meaning
POSITIVE GREEN Anti-HIV antibodies PRESENT in serum
NEGATIVE COLORLESS No anti-HIV antibodies detected
Positive ELISA requires confirmatory testing (Western blot or nucleic acid test)

Lab Controls

Control Expected Result Purpose
PBS (negative) Colorless Confirms no false positives
+ Control Green Confirms test is working

Test-Taking Points

  • ELISA = Enzyme-Linked Immunosorbent Assay
  • Indirect ELISA detects antibodies; Direct detects antigens
  • GREEN = positive (antibodies detected)
  • Colorless = negative (no antibodies)
  • Secondary antibody has enzyme attached (peroxidase)
  • Enzyme converts substrate to colored product
  • Washing steps are critical to remove unbound material
  • ELISA is a screening test - positive results need confirmation
  • Uses: HIV, hepatitis, West Nile virus, syphilis, COVID-19
