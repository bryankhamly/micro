Epidemiology - Core Concepts

Key Concepts for the Test

Essential Epidemiology Terms

Term Definition
Endemic Steady frequency of disease in a region over time
Epidemic Excessive cases in an area compared to normal
Pandemic Epidemic spread across continents (e.g., COVID-19)
Index case First identified case in an outbreak
Carrier Person who harbors and transmits pathogen but shows no symptoms
Fomite Inanimate object that transmits pathogens (doorknob, phone)
Vector Living carrier (insect/rodent) that transmits pathogen

Two Types of Epidemics

Feature Common Source Host-to-Host (Propagated)
Source Single contaminated source (food, water) Infected individuals
Onset Rapid; many cases at once Gradual; cases increase over time
Curve shape Sharp peak, rapid decline Gradual rise, slower decline
Control Remove contaminated source Isolation, vaccination
Example E. coli outbreak from lettuce Influenza, COVID-19

Herd Immunity

When a sufficient portion of a population is immune, outbreak is unlikely because spread is limited.
  • Protects those who cannot be vaccinated (infants, immunocompromised)
  • Higher R0 (reproduction number) = higher threshold needed
Disease Herd Immunity Threshold
Measles 92-95%
Polio 80-86%
COVID-19 (original) 60-70%
Influenza 33-50%

Test-Taking Points

  • Endemic = constant presence; Epidemic = excessive cases; Pandemic = global spread
  • Index case = first identified case - epidemiologists trace backward from this
  • Common source epidemic = sharp peak (contaminated food), propagated = gradual rise (person-to-person)
  • Herd immunity protects unvaccinated individuals when enough population is immune
  • R0 = basic reproduction number (average # infected by one person)
  • Fomite = inanimate object; Vector = living carrier
  • Higher R0 = higher herd immunity threshold needed
